# Получить список складов

**URL** : `/storages`

**Method** : `GET`

## Success Responses

**Code** : `200 OK`

**Content** :

```json
[
    {
        "name": "Storage name 1"
    },
    {
        "name": "Storage name 2"
    },
    {
        "name": "Storage name 3"
    }
]
```

## Error Responses

**Code** : `500 Internal Server Error`
